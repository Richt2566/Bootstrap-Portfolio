# Bootstrap Portfolio

This repository contains a simple portfolio design using Bootstrap to demonstrate mobile responsivness. 

## Objectives

* Use Bootstrap to witness the simplicity of its mobile respoinsive behavior. 
* Life without media queries is usefull but slightly limiting. 

## Built With

* Html5
* CSS3
* Bootstrap 3.3.6
